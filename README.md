# muchsyncrs

I think [muchsync] is awesome. Lets reimplement it in Rust.

## Goal / Compatibility

The goal is **not** to be compatible to "[muchsync] the protocol", but to fulfill
the same purpose as "[muchsync] the tool".

This means: **No Compatiblity!!!**
